# Иконка сайта

На этой вкладке можно вставить теги с указанием иконки сайта, например: `<link rel="icon" type="image/png" href="favicon.png" />`.

Вы можете воспользоваться специальными генераторами иконок (ссылка на один из них предлагается по умолчанию), либо написать нужные теги самостоятельно.
