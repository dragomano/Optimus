## Главная страница {#homepage}
---
<Steps>
1. ### Заголовок главной страницы форума {#forum-homepage-title}

    Значит ровно то, что написано. Измените и проверьте результат.

2. ### Описание форума {#the-forum-description}

    Аналогично, см. выше.

3. ### Ключевые слова, соответствующие форуму {#meta-keywords-associated-with-forum}

    Эта опция перемещена сюда из встроенных настроек форума, чтобы структурировать родственные настройки в одном месте. Просто укажите набор слов, характеризующих тематику вашего форума.
</Steps>

## Страницы тем и разделов {#topic--board-pages}
---
<Steps>
1. ### Добавлять название форума к заголовкам разделов {#add-forum-name-to-board-titles}

    Выберите из 3-х вариантов.

    Допустим, ваш форум называется «Тайный клуб грибников из Суворово», и есть у вас раздел с названием «Грибные места». Когда вы зайдете внутрь раздела и посмотрите на заголовок страницы, результат будет зависеть от выбранного вами варианта опции:

    | Вариант                  |                     Результат                       |
    | :----------------------- | :-------------------------------------------------: |
    | Не добавлять             |          Без изменений (`Грибные места`)            |
    | Перед заголовком раздела | `Тайный клуб грибников из Суворово - Грибные места` |
    | После заголовка раздела  | `Грибные места - Тайный клуб грибников из Суворово` |

2. ### Добавлять заголовок раздела и название форума к заголовкам тем {#add-board-title-and-forum-name-to-topic-titles}

    Выберите из 3-х вариантов.

    Представим, что есть у вас в разделе «Грибные места» тема с названием «Ближайшая сходка»:

    | Вариант               |                              Результат                                 |
    | :-------------------- | :--------------------------------------------------------------------: |
    | Не добавлять          |                  Без изменений (`Ближайшая сходка`)                    |
    | Перед заголовком темы | `Тайный клуб грибников из Суворово - Грибные места - Ближайшая сходка` |
    | После заголовка темы  | `Ближайшая сходка - Грибные места - Тайный клуб грибников из Суворово` |

    Не нужны вам эти опции? Не включайте, если не нужны. Поехали дальше.

3. ### Использовать отрывок первого сообщения темы в качестве мета-тега `description` {#display-the-topic-first-message-snippet-as-the-meta-tag-description}

    Вариант для ленивых, если вам не хочется вручную придумывать/заполнять описания тем.

4. ### Разрешить отдельное поле для описания темы {#allow-a-separate-field-for-the-topic-description}

    Вариант для дотошных, можно создавать описание для каждой темы отдельно. Не забудьте выставить в правах доступа разрешения для тех групп пользователей, которым хотите разрешить заполнение описаний.

5. ### Разрешить отдельное поле для тегов темы {#allow-a-separate-field-for-the-topic-tags}

    Аналогично, для заполнения тегов. Сами они заполняться не будут. Не забудьте выставить в правах доступа разрешения для тех групп пользователей, которым хотите разрешить заполнение тегов.

6. ### Отображать блок с тегами над первым сообщением темы {#display-a-block-with-tags-above-the-first-post-of-the-topic}

    Заполнили теги? Хорошо! А отобразить их хотите? Тогда включайте.

7. ### Отображать теги в списках тем внутри разделов {#display-tags-in-topic-lists-within-boards}

    Очень красиво будут отображаться, проверьте.

8. ### Использовать цветные теги {#use-color-tags}

    А если это включить, теги станут ещё красивее.

9. ### Максимальное количество тегов в одной теме {#maximum-number-of-tags-per-topic}

    Мера контроля, чтобы особо ретивые не смогли перегрузить ваш форум тегами.
</Steps>

## Дополнительные настройки {#additional-settings}
---
<Steps>
1. ### Отображать страницу 404 для несуществующих областей форума {#display-a-404-page-for-non-existent-forum-areas}

    Просто проверьте реакцию форума на несуществущие области без этой опции и с ней, и выберите нужный вам вариант.

2. ### Отображать страницу 403/404 для недоступных/несуществующих разделов и тем {#display-a-403404-page-for-inaccessiblenon-existent-boards-and-topics}

    Просто проверьте реакцию форума на несуществущие страницы разделов и тем без этой опции и с ней, и выберите нужный вам вариант.

3. ### Вести статистику поисковых запросов {#enable-logging-of-search-terms}

    Если включите, в отдельную таблицу в базе данных будут добавляться фразы, которые ваши пользователи ищут на вашем форуме.
</Steps>
