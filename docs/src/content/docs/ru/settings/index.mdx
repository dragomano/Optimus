После успешной установки мода вы автоматически попадёте на страницу настроек Optimus. Также туда можно попасть, перейдя в раздел «Конфигурация => Поисковая оптимизация» в админке SMF.

Главный секрет успешного использования и освоения Optimus — не трогать то, что работает. Если вас устраивают текущие настройки, не экспериментируйте, продолжайте использовать их. Экспериментируйте только на локальном или тестовом сайте.

Если смысл какой-то опции непонятен, почитайте её описание в этой документации. Если непонятен какой-то термин (мета-тег, `description` и т. д.) — погуглите.

Настройки модификации разделены на разделы-вкладки, для удобства использования. В верхней части каждой вкладки есть ссылка на полезную справочную информацию.
